### 이미지 시점확인 솔루션

온더픽셀은 2019년 3건의 이미지 시점확인 처리 기술에 대한 특허를 출원하였습니다. 래스터 이미지에 대한 무결성과 진본성을 위해 이미지 자체적으로 이를 증명할 수 있는 기술입니다.
